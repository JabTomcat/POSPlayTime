This is a little PWA that I made for my son to play as a waiter and take orders. It allows him to create an invoice and then press a paid button to "collect money".
- Everything is built to run cached in the local browser.
- Designed to work on an iPad, but works in most browsers/devices.
- Can go to an admin page to allow editing of the items.
- A menu page is available via a QR code on the main page for customers to get access to a menu on their devices.

You can access the site via the below URL:

https://jabtomcat.github.io/POSPlayTime/
